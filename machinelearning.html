Notable Accomplishments in Machine Learning:
Person/Company Date Short Description
Fei-Fei Li 2009 Fei-Fei Li created ImageNet, a large visual database that stores annotated photographs intended for computer vision research. ImageNet has been an instrumental tool for advancing computer vision and deep learning research. For many, ImageNet was the catalyst for the AI boom of the 21st century.
Warren McCulloch
Walter Pitts 		1943 Warren McCulloch and Walter Pitts developed a mathematical model that imitates the functioning of a biological neuron. They called it the “artificial neuron” which is considered to be the first neural model invented.
Google 2016 Google’s AlphaGo program became the first Computer Go program to beat an unhandicapped professional human player using machine learning and tree search techniques. AlphaGo is a computer program that plays the board game Go, a Chinese strategy board game, against human competitors. Creating a learning algorithm that can beat a human player at strageic games is a measure of A.I. development and progress.
